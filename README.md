# vgmcharts
Explore trending video game music

## First things first: create a .env file

Find the terminal command appropriate for your operating system below and copy the example `.env-example` into a local `.env` you can then edit with the credentials we provide. Contact us for the deets!

Unix / macOS (bash / zsh):

```sh
cp .env-example .env
# edit .env with your editor by typing `code .env`, `vim .env`, etc.
```

PowerShell (Windows):
```powershell
Copy-Item .env-example .env
# then edit .env with Notepad or your preferred editor
```

The gitignore in this project is already set up to never commit your `.env` file; please keep secrets out of version control (aka git).
